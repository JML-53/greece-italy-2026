📶 2. Airalo eSIM (data abroad)

Use the Eurolink (Europe) plan — it covers every country on this trip: Greece, Italy, Croatia, Slovenia, and Montenegro.

  1. On Wi-Fi, install Airalo from the Play Store; sign in or create an account.
  2. Store → Regional → Europe (Eurolink). Pick a data size (≈5–10 GB is plenty for maps and messaging over the remaining weeks). Before paying, confirm the coverage list includes Greece, Italy, Croatia, Slovenia, and Montenegro. Buy.
  3. My eSIMs → tap the new eSIM → Install or share → Start installation → follow the prompts. Don't leave the screen until it finishes (can take a few minutes).
  4. Phone Settings → Connections → SIM manager: set the Airalo eSIM as Mobile data and turn Data roaming ON for the Airalo line.
  5. Turn the Verizon line OFF — SIM manager → switch the line off (or turn its data roaming off and don't call or text on it). Verizon's overseas pass is $12/day per line and kicks in on any call, any text (even incoming), or any data — all of which Airalo now covers. Do talk & text over WhatsApp, iMessage/FaceTime, or email instead. Everyone on Verizon should do this.
  6. It activates once it grabs a local network (may take a few minutes; toggle Airplane mode to nudge it). Test by loading a page with Wi-Fi off.

US number & 2FA: with the Verizon line off you won't receive SMS security codes. If you need one, toggle that line's roaming on for the day (accept the $12), grab the code, then turn it off again — or move that login to app/email codes.

Airalo is data-only — no calls or texts on it. If data won't connect, open the eSIM's in-app instructions and check the APN. Running low? Top up in the app.
