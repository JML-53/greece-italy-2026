# 📱 Phone Setup — Teri's Phone (mid-trip)

**Why now:** Joe's phone was left on the Delta aircraft in Rome (Lost Report **#2734861**). Until it's recovered or replaced, Teri's phone is the group's device for reaching Claude, getting data abroad, and — most urgently — recording the GPS route.

Do these on hotel or ship Wi-Fi. Assumes Teri's phone is a Samsung Galaxy (Android); iPhone steps are nearly identical.

---

## ✅ Checklist 1 — Reach Claude & the trip on Teri's phone

Goal: ask trip questions on the go, signed in as Joe.

- [ ] Install the **Claude** app from the **Google Play Store** (search "Claude by Anthropic").
- [ ] Open it and **Sign in** with Joe's login: **jmlmail53@gmail.com**.
  - ⚠️ If it asks for a verification code, choose the **email** option — a text or authenticator code would go to the lost phone.
- [ ] Bottom menu → **Projects** → open **Travel** (the Greece & Italy project).
- [ ] Tap **New chat** inside the project and ask anything — e.g. *"What's on for tomorrow?"* or *"Where's the Vatican tour meeting spot?"* The project already knows the itinerary.

**Note on site edits:** Asking questions works fine on the phone. But *editing and publishing the trip website* runs in a Cowork session tied to the PC folder, so **site changes must be made from the computer**. From the phone you can still ask questions and jot down changes you want made later.

## ✅ Checklist 2 — Airalo eSIM (data abroad)

Use the **Eurolink (Europe)** plan — it covers every country on this trip: **Greece, Italy, Croatia, Slovenia, and Montenegro.**

- [ ] On Wi-Fi, install **Airalo** from the Play Store; sign in or create an account.
- [ ] **Store → Regional → Europe (Eurolink)**. Pick a data size (**≈5–10 GB** is plenty for maps and messaging over the remaining weeks).
- [ ] Before paying, confirm the coverage list includes **Greece, Italy, Croatia, Slovenia, Montenegro**. Buy.
- [ ] **My eSIMs** → tap the new eSIM → **Install or share → Start installation** → follow the prompts. Don't leave the screen until it finishes (a few minutes).
- [ ] **Settings → Connections → SIM manager**: set the **Airalo** eSIM as **Mobile data**, and turn **Data roaming ON** for the Airalo line.
- [ ] Keep Teri's **Verizon** line for calls/texts, but turn its **Data roaming OFF** to avoid charges.
- [ ] It activates once it grabs a local network (toggle Airplane mode to nudge it). **Test:** load a webpage with Wi-Fi off.

**Notes:** Data-only — no calls/texts on the Airalo line. If data won't connect, open the eSIM's in-app instructions and check the APN. Top up in the app if you run low.

## ✅ Checklist 3 — Polarsteps GPS tracking (make THIS the tracker)

> **Do this soon.** Polarsteps records the route from **only one phone at a time**, and Joe's phone was the default tracker — so tracking likely **stopped** when it was lost. Turning Teri's phone into the active tracker recaptures the rest of the trip.

- [ ] Install **Polarsteps** from the Play Store and open it.
- [ ] **Sign in as Joe** (same login as Claude) — the "Greece & Italy 2026" trip is already there.
  - *Alternative:* use Teri's own Polarsteps account if Joe adds her as a **Travel Buddy** and she accepts the invite.
- [ ] Open the **Greece & Italy 2026** trip → **trip settings** → **Travel Tracker** → set **this phone** as the tracking device.
- [ ] When prompted for location, choose **Allow all the time** and enable **Precise location**.
- [ ] Exempt Polarsteps from **battery optimization** so it keeps logging in your pocket.
- [ ] Confirm the tracker toggle is **ON**. It logs offline and syncs on Wi-Fi — no mobile data needed for tracking.
- [ ] *Optional backup:* leave **Google Timeline** on (Settings → Location → Location services → Timeline).

**Notes:** Tracking uses ~4%/day battery — carry the power bank on long port days. Everyone in the trip can still add photos and daily notes.
